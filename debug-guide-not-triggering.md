# 假设

- 假设 1：`GuideManager` 根本没有被正确加载或 `_ready()` 没有执行。

- 假设 3：引导状态被判定为 `disabled`、`unknown` 或已完成，导致 `start_default_guide_if_needed()` 直接返回。
- 假设 4：引导配置 `guide_flows.json` 已加载，但默认引导 ID、步骤或触发条件不满足，导致没有任何可显示步骤。
- 假设 5：引导步骤实际已进入，但在 `_refresh_current_step_display()` 或 `_show_step_overlay()` 阶段被“场景未就绪 / overlay 未挂载 / step 被跳过”吞掉。
